Ride It Analysis Dashboard

Project Overview

Ride It Analysis is a Power BI dashboard project focused on analyzing ride booking operations, driver performance, regional trends, and cancellation behavior.
The project helps identify key operational insights such as ride demand patterns, driver efficiency, country-level performance, and cancellation trends.

Tools Used

* Excel
* Power BI
* Power Query
* DAX
* Data Cleaning
* Data Modeling

Dashboard Pages

1. Executive Overview

This page provides a high-level summary of the business.

Key metrics:

* Total Offers
* Total Bookings
* Total Completed Rides
* Acceptance Rate
* Completion Rate

Insights:

* Overall ride demand trend
* Conversion funnel performance


2. Driver Analysis

This page analyzes driver behavior and contribution.

Key metrics:

* Marketing impact on rides
* Driver rating vs ride volume
* Driver experience contribution
* Gold level segmentation

Insights:

* High-rated drivers contribute more rides
* New and experienced drivers contribute almost equally


3. Regional Analysis

This page compares country-level performance.

Key metrics:

* Ride funnel by country
* Acceptance vs completion rates
* Cancellation patterns
* Service type distribution

Insights:

* Germany has both Taxi and PHV services
* Spain mainly operates Taxi rides


4. Time Analysis

This page tracks business performance over time.

Key metrics:

* Driver cancellation rate
* Passenger cancellation rate
* Total cancellations
* Ride funnel trend over time
* Driver registration trend
* Cancellation trend over time

Insights:

* Booking trends over time
* Cancellation behavior patterns
* Driver onboarding growth


Key Business Insights

* Acceptance rate is lower than completion rate, indicating strong ride completion after booking.
* Driver ratings positively influence ride contribution.
* Germany has diversified service types, while Spain is more focused on Taxi rides.
* Cancellation patterns vary significantly across regions.
